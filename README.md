## Features

- Product catalog with categories
- Shopping cart functionality
- User registration & authentication
- Order management
- Responsive product pages

## Getting Started

```bash
git clone https://github.com/skyson-ai/Skyson-shop.git
cd Skyson-shop
pip install -r requirements.txt
python app.py
```
